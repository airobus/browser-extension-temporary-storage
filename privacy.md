# 隐私政策

## 数据收集
- 本插件不收集任何个人信息
- 所有笔记内容仅存储在用户本地
- Notion 集成仅在用户主动配置后使用

## 数据存储
- 使用 Chrome Storage API 本地存储
- 不上传任何数据到第三方服务器
- Notion 集成仅将用户指定内容发送到用户自己的 Notion 页面

## 权限说明
- storage: 用于本地存储笔记内容
- contextMenus: 用于右键菜单功能
- api.notion.com: 用于 Notion 集成功能
